All commands are accessible via the Command Palette (`Ctrl+Shift+P` on Windows/Linux, `Cmd+Shift+P` on macOS). Type "SEO:" to filter to extension commands.

| Command | Description | Plan Required |
|---|---|---|
| SEO: Run SEO Check | Runs SEO and accessibility checks on the currently open HTML/JSX file | Free |
| SEO: Generate Schema Markup | Opens a guided form to generate JSON-LD structured data | Free |
| SEO: Find Unused Imports | Scans the current TypeScript/JavaScript file for unused imports | Free |
| SEO: Optimize Image | Right-click an image in the Explorer → compresses and converts to WebP | Free |
| SEO: Run Full Audit (Live URL) | Runs SEO and accessibility checks against a live dev server URL | Free |
| SEO: Run Performance Audit | Runs a full Lighthouse audit and returns Core Web Vitals scores | Pro |
| SEO: Open Full Dashboard | Runs all checks and opens a visual score dashboard | Pro |
| SEO: Enter License Key | Saves and validates your Pro license key | — |
| SEO: Check License Status | Shows whether you have an active Pro license | — |
