## Installation

Search for **"SEO Insight Engine"** in the VS Code Extensions Marketplace (`Ctrl+Shift+X`), or install directly from the command palette:

```
ext install seo-insight-engine
```

No configuration required. Free features work immediately after installation — no account, no sign-in, no setup.

## System Requirements

- Visual Studio Code 1.80 or later
- Node.js 18 or later (required for performance auditing)
- Internet connection (required for license validation and AI features only)
